# 🔍 Auditorías Éticas y de Seguridad

## Evaluaciones de Privacidad e Interfaz
* [cite_start]**Privacidad por Diseño (Privacy by Design):** Establece que la protección de datos debe incorporarse dentro de la arquitectura del sistema y no implementarse únicamente como medida correctiva posterior[cite: 154].
* [cite_start]**Auditoría de Patrones Oscuros (Dark Patterns):** Se deben identificar las interfaces orientadas a manipular el comportamiento de los usuarios mediante configuraciones engañosas o procesos poco transparentes[cite: 155].
