## Dimensión Ambiental (Sostenibilidad TIC)
[cite_start]Los profesionales deben diseñar soluciones responsables y sostenibles[cite: 41]. Esto exige enfocarse en:
* [cite_start]Gestión de e-waste[cite: 42].
* [cite_start]Eficiencia energética[cite: 43].
* [cite_start]Digitalización responsable[cite: 44].
* [cite_start]Uso sostenible de hardware[cite: 45].

## Infraestructura y Desarrollo Verde
* [cite_start]**Sistemas y Redes:** El cuidado ambiental requiere virtualización, cloud computing eficiente, data centers verdes y el uso de energías renovables[cite: 33, 34, 35, 36, 37].
* [cite_start]**Software:** El desarrollo de software sostenible exige un código eficiente, optimización de recursos, green coding y un ciclo de vida del software planificado[cite: 28, 29, 30, 31, 32].
